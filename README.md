# 🤖 Arduino data faker

A cross-platform Python environment to simulate Arduino serial data. This project allows you to train **pyserial** data analysis and robotics logic on Linux, macOS, and Windows without needing physical hardware.

---

### 🚀 Quick Start

#### **1. Environment Setup**
Open your terminal (Bash on Linux/Mac, or PowerShell on Windows) and run:

```bash
# Clone the repository
git clone [https://github.com/yourusername/virtual-arduino-sim.git](https://github.com/yourusername/virtual-arduino-sim.git)
cd virtual-arduino-sim

# Create a virtual environment
python -m venv venv

# Activate the environment
# On Linux/macOS:
source venv/bin/activate
# On Windows:
.\venv\Scripts\activate

# Install dependencies
pip install pyserial
```
